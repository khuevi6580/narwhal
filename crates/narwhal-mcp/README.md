# narwhal-mcp

Model Context Protocol (MCP) server.

Exposes narwhal's configured connections to AI agents over JSON-RPC 2.0 stdio. Driver-agnostic via `narwhal-driver-registry`.

Part of the [narwhal](../../README.md) workspace. See [`docs/ARCHITECTURE.md`](../../docs/ARCHITECTURE.md) for the layered crate map.
