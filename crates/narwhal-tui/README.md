# narwhal-tui

Ratatui-based UI layer.

Layout, theme, key translation and widgets (editor, results, sidebar, history, snippets, wizard, help, row detail). Reads domain models by reference; never mutates them.

Part of the [narwhal](../../README.md) workspace. See [`docs/ARCHITECTURE.md`](../../docs/ARCHITECTURE.md) for the layered crate map.
