# narwhal-commands

Stateless command and helper modules.

Command parsing and dispatch, completion engine, export pipeline (CSV/JSON/TSV/table/INSERT), connection wizard, snippet store, DDL helpers, EXPLAIN, inline cell edit, statement extraction, meta queries. Self-contained — does not reach back into `narwhal-app`.

Part of the [narwhal](../../README.md) workspace. See [`docs/ARCHITECTURE.md`](../../docs/ARCHITECTURE.md) for the layered crate map.
